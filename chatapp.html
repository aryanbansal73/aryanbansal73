ChatApp Project

About the Project

ChatApp is a real-time chat application that allows users to communicate with each other through text messages. It provides a user-friendly interface and supports features like one-on-one chat, group chat, and file sharing.

Features

  • Real-time Messaging
  • One-on-One Chat
  • Group Chat
  • File Sharing

Technologies Used

This project was developed using HTML, CSS, JavaScript, and Node.js for the server-side component.
